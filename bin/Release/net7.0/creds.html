What to do next

These credentials must be pasted into a CSV file located here:

{{filePath}}creds.csv

The headings within creds.csv must be as follows. Any deviation from this and the insecure defaults will be used!!!

userName,encPassword,adhocSpaceName,confBridgePrefix

Headings in the creds.csv file

  • userName - The username this application will use to log into CMS. From the Credentials section above
  • encPassword - An encrytpted password string used by this application to log into CMS. From the Credentials section above
  • adhocSpaceName - A text string that will replace the long numeric string space name
  • confBridgePrefix - A prefix you have configured on CUCM conference bridge configuration. Any conference with this prefix will have its name change to whatever you have set for adhocSpaceName. see the Configuring CUCM for this web app on the main page
